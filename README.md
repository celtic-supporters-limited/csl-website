# CSL Website

Public website for Celtic Supporters Limited, built with Next.js 14 (App Router), Tailwind CSS, Stripe, Supabase, Resend, and Zoho CRM.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Hosting**: Vercel
- **Payments**: Stripe
- **Database + Auth**: Supabase (EU region)
- **Email**: Resend
- **CRM**: Zoho CRM (EU data centre)
